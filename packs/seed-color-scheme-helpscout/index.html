Install

npm install seed-color-scheme-helpscout --save

Dependencies

  • seed-color-scheme

Usage

The Help Scout color scheme pack consists of all the colors and shades in our color palette.

Use the _color() mixin (provided by seed-color-scheme) to reference a color from this pack.

example.scss
.element {
  background: _color(blue, 500);
}

example.css
.element {
  background: #3197D6;
}
