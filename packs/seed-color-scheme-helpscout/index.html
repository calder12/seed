    Pack: seed-color-scheme-helpscout

    Color scheme (Help Scout)

    View Source

    A series of color scheme variables with Help Scout’s color palette.

    Install

    npm install seed-color-scheme-helpscout --save

    Dependencies

    • seed-color-scheme

    Usage

    This pack consists of all the colors and shades in Help Scout’s color palette.

    The _color() mixin (provided by seed-color-scheme) is used to reference a color from this pack.

    example.scss
    .element {
  background: _color(blue, 500);
}

    example.css
    .element {
  background: #3197D6;
}
