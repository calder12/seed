Install

npm install seed-position --save

Dependencies

  • seed-breakpoints
  • seed-publish

Usage

All of the following classes support responsive modifiers.

Class Position
u-pos-initial initial
u-pos-inherit inherit
u-pos-absolute absolute
u-pos-fixed fixed
u-pos-relative relative
u-pos-static static

Configurations

Below are the default Sass configuration variables for seed-position. The variables can be found in the packâ€™s _config.scss file.

// Namespace
$seed-position-namespace: u-pos !default;

// Positions
$seed-positions: (
  absolute: absolute,
  fixed: fixed,
  inherit: inherit,
  initial: initial,
  relative: relative,
  static: static
) !default
